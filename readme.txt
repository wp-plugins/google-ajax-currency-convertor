== Description ==
Currency Convertor that uses Google Calculator and update the results with AJAX

== Installation ==
1. Upload plugin directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto the 'Widgets' menu in Wordpress and add Google AJAX Currency Convertor to your sidebar

== Frequently Asked Questions ==
= How to change the list of currencies = 
Goto Stetings -> Currency List
